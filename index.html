Left Child Right Sibling Representation

Default maximum degree is set to 5. Let's say we are at Node 1 and there are 4 possible children Node 4, Node 5, Node 6, Node 7 each of these nodes would need a certain number combination
with Alt to jump to.

Alt + 1 would take you to Node 4.
Alt + 2 would take you to Node 5.
Alt + 3 would take you to Node 6.
Alt + 4 would take you to Node 7.

And if there is only one child, then Alt + 1 will act as redo.

About the Project

Each individual stroke made by the user in the canvas is recorded and that specific stroke's data is turned into a node and added to the tree. As the user makes each new stroke the visual representation of the left-child right-sibling tree updates in real time and shows the structure of the tree being formed.

The user also has access to undo and redo functions which are mapped to specific key combinations. Unlike conventional undo-redo which is implemented using stack, this project has implemented the functionality using a Tree Structure.
